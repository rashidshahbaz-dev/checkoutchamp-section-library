# Hero V1

Reusable CheckoutChamp hero section with product gallery, thumbnail navigation, offer cards, stock line, CTA, and mini accordion.

## Files

- `index.html` = full preview
- `style.css` = component CSS
- `script.js` = gallery, offer select, and accordion JS

## Copy to CheckoutChamp

Copy the `.coc-hero` section HTML, then add `style.css` and `script.js`.

## Variables used

- `--coc-font-main`
- `--coc-color-text`
- `--coc-color-heading`
- `--coc-color-white`
- `--coc-color-accent`
