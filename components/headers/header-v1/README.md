# Header V1

Reusable CheckoutChamp header component with announcement topbar and centered logo.

## Files

- `index.html` = full preview
- `style.css` = component CSS
- `script.js` = optional JS
- `README.md` = usage notes

## CheckoutChamp usage

Copy the component HTML into the page.

Copy `style.css` into page CSS.

Copy `script.js` only if needed.
