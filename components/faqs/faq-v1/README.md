# FAQ V1

Reusable CheckoutChamp FAQ accordion component.

## Files

- `index.html` = full preview
- `style.css` = component CSS
- `script.js` = one-open-at-a-time accordion JS

## CheckoutChamp usage

Copy the `.coc-faq` section HTML, then add `style.css` and `script.js`.

## Behavior

- All FAQ items closed by default
- Only one FAQ item opens at a time
- Clicking an open item closes it
